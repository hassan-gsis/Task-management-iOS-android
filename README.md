# Flutter-Task-Manager



This is a daily task manager application project created in flutter. Install this application from [Google Play Store](https://play.google.com/store/apps/details?id=bornomala.todo) or Get it from [GitHub Release](https://github.com/hassan-gsis/Task-management-iOS-android).

## Getting Started

Step-1: Download or clone this repository -

    git clone https://github.com/hassan-gsis/Task-management-iOS-android

Step-2: Go to project root and execute the following command in console to get the required dependencies -

    flutter pub get 
    
Step-3: Run flutter project -

    flutter run
